## Bagian 1: Unduh Android Studio

1. Buka browser dan kunjungi situs resmi [Android Developer](https://developer.android.com/studio).
2. Klik tombol **Download Android Studio**.
3. Baca dan setujui *Terms and Conditions*, lalu unduh *installer* sesuai dengan sistem operasi yang digunakan.

## Bagian 2: Proses Instalasi
Langkah instalasi bisa disesuaikan dengan sistem operasi yang digunakan:

### Untuk Pengguna Windows:
1. Jalankan file installer `.exe` yang sudah diunduh.
2. Ikuti *Setup Wizard*. Pada bagian **Choose Components**, pastikan opsi **Android Virtual Device** dicentang.
3. Klik **Next** hingga proses instalasi selesai dan klik **Finish** untuk menjalankan Android Studio.

### Untuk Pengguna Ubuntu / Linux:
1. Ekstrak file `.tar.gz` yang diunduh ke direktori yang diinginkan (misalnya `/usr/local/` atau `/opt/`).
2. Buka terminal, lalu navigasi ke direktori *bin* di dalam folder hasil ekstraksi:
   ```bash
   cd /path/to/android-studio/bin
